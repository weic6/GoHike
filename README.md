# GoHike

Go, Hikes! Explore your next hiking spots! This full stack web app was built with Express and Reacts.

## Run guide

```shell
cd client && npm i
```

```shell
cd server && npm i
```

```shell
cd <parent project folder> && npm i
npm start
```
